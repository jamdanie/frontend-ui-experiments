Escalation Logic

The system asks first, then escalates based on risk and non-response.

Stages

  1. Nudge: boundary or hazard warning
  2. Check-in: “Are you still active?”
  3. Intent mismatch: prolonged inactivity + deviation + offline
  4. Notify POC: last known location + time + trail segment
  5. Responder handoff (Phase 3): verified partnerships only
Privacy note: No identity is shown to other users; emergency profile is shared only during escalation.
